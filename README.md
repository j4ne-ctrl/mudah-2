# LANGKAH KERJA PROGRAM GANJIL GENAP
Program ini berfungsi untuk menentukan angka apakah yang dimasukkan user merupakan ganjil atau genap.

Dalam program ini, user diminta untuk memasukkan angka. Program akan memeriksa apakah angka tersebut habis dibagi dua. Jika ya, maka angka itu genap. Jika tidak, berarti ganjil. Lalu program akan menampilkan hasilnya di output
